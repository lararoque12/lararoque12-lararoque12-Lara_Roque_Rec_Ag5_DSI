## 🖩 Calculadora de Conta de Luz

* Lingugagem Utilizada: Python🐍 

* Fórmula utilizada para o cálculo:
    consumoMensal = (watts * tempo * 30) / 1000
    valorFixo = (consumoMensal * 0.5)
    
 • No programa é solicitado inicialmente qual é a demanda proposta
    pelo usuario, assim o mesmo tendo que digita-lá e logo após é 
    demonstrado o seu consumo de watts e o valor de sua conta de energia elétrica.
